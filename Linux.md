1. What is Linux?
- Linux is an open-source operating system based on Unix. It is known for its stability, security, and flexibility.

2. What is the difference between Linux and Unix?
- Unix is a proprietary operating system, while Linux is open-source.
- Linux is a Unix-like system but is developed independently and is more flexible.

3. What is a Linux kernel?
- The kernel is the core part of the Linux operating system that manages hardware, system resources, and communication between software and hardware.

4. What is the purpose of chmod command?
- The chmod command is used to change the permissions of a file or directory (e.g., read, write, execute). Example: chmod 755 file.txt

5. What is the ps command used for?
- The ps command displays information about running processes on the system. For example, ps aux shows all processes.

6. What are symbolic links in Linux?
- A symbolic link (symlink) is a file that points to another file or directory, acting as a shortcut. Created with the ln -s command.

7. How do you find a file in Linux?
- You can use the find command to search for files in a directory hierarchy: find /path/to/search -name "filename"

8. What is the grep command used for?
- The grep command is used to search for specific patterns or text in files. Example: grep "pattern" filename

9. What is the top command used for?
- The top command displays a dynamic, real-time view of system processes, CPU usage, memory, and other system information.

10. How do you check disk space in Linux?
- Use the df command to check disk space: df -h
